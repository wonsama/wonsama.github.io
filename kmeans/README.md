# k-means 란 ?

steemjs 를 활용하여 나의 팔로워 정보중 명성과 스파 사이의 상관관계를 k-means를 통해 볼 수 있습니다.

# k-means 사용방법 ( with steemit )

### TEST LINK

https://wonsama.github.io/kmeans

### 1. 그리기
1. 아이디 입력
1. (옵션) 최소 스파 설정 0 이상
1. (옵션) 클러스터 수 3이 제일 적절함.
1. (옵션) 제외 아이디(컴마로 구분)

### 2. 다시그리기 
1. 현재 그려진 정보를 기반으로 다시그려 주도록 합니다.(초기 group의 point 재설정을 위함)

### 3. (동그라미) 마우스 오버/아웃 
1. 현재 위치의 대상의 아이디, 스파, 명성을 확인할 수 있다.
1. 마우스 아웃 시 사라짐

### 3. (동그라미) 클릭
1. 하단에 현재 위치의 대상 정보를 표기
1. 다른 정보를 클릭하기 전까지 고정되어 하단 표기 됨

# 유의사항
