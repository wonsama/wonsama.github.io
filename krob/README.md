# krob(kr-observer) 개요

* KR 트랜드의 최신글을 자동 갱신하여 확인하실 수 있습니다.(10초단위)
* 태깅을 통한 어뷰징을 어느정도 미리 감지 할 수 있음 (한글없음, 블랙리스트)

# 특징

* 10초 단위로 정보를 갱신하여 특정 tag의 trends를 지속적으로 확인 할 수 있습니다.
* 15SP 이하는 배경색이 회색
* 주소창 뒤에 ?tag=kr-dev 이런식으로 태그를 넣어주면 해당 태그 관련글 보여줌, 기본 kr
* 문의하기 - 0.001 SBD 전송을 통해 궁금한 것을 문의 하실 수 있습니다. (답변시 환불됨)
* 후원하기 - 정말정말 맘에 들었다하면 1SBD 후원을 하실수도 있습니다. 감사합니다 ^^

예시)

# 태그 구분 

* (BLUE) kr태그 - kr로 시작하는 태그
* (RED) 블랙리스트 - 기존 autovot 에서 kr-guide로 걸린 계정은 blacklist라고 태그가 보여짐 ( krop_black.js 에서 BLACK_AUTHORS 참조 )
* (RED) 한글없음 - 제목과 본문에 한글이 없으면 한글 없음이라 표기됨
* (YELLOW) jjangjjangman - 1일 1회 오치님께서 보팅을 주는 태그
